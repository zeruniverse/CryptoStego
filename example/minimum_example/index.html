CryptoStego

A JS library for steganography with encryption - Hide text in an image with encryption and obfuscation.
This library provides two modes of steganography - least significant bit mode and DCT mode.
least significant bit mode gives result more similar to the original image while DCT mode gives more robustness to image compression

How to use example above

Bob selects an image, input the message and click 'hide my message to this image'. He then downloads the result image and sends it to Alice.

Alice opens this webpage, selects the image that Bob sent to her, (Inputs password if Bob set one), and then click 'Read My MSG from Image'. She sees what Bob wants to tell her.

Features

Obfuscation - Initialization with random data.

Non-linear storage - Information stored by bits. Location of each bit is determined by hash function

Security - Password decides the sha512-based hash function. There's no simple way to check whether password is correct. If incorrect password is input, either you get wrong data or UTF8_decode fails to execute.
